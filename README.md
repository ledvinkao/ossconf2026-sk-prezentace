This repository relates to the presentation given at the OSSConf2026 conference (see https://ossconf.fri.uniza.sk/). A Quarto HTML document is the most important, serving as the presentation material, as well as for the demonstration of working with raster geodata in R. The end part deals with the parallelization of saving individual raster layers to GeoTIFF files. However, those who are interested in details are referred to what is written in the accompanying proceedings. The processed geodata can be found at https://cds.climate.copernicus.eu/.
